***Welcome to TTAgP 1.0 (Tumor T-cell Antigen Predictor) is a roboust program to predict tumor peptides with a friendly interface.

Steps to run TTAgP 1.0:

1- Extract all the TTAgP 1.0 files

2- Install WinPython 3.6.6.2Qt5-64 bit (*) available at: https://winpython.github.io/

3- Copy and paste the TTAgP 1.0 files into the installed WinPython software

4- Locate and run the TTAgP 1.0.py file
